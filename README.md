# Minomaly engine

# Requirements

- [C++17 compatible compiler](https://en.cppreference.com/w/cpp/compiler_support) (e.g. MSVC 19+)
- [CMake 3.0+](https://cmake.org/)

# Initializing CMake

```
cmake -H. -Bbuild
```

# Building

```
cmake --build build
```

# Running the example

```
build\Debug\app.exe # on Windows
```
